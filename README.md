# 项目简介

名称：vue-h5

简述：【 基于koa2+vue2全家桶+webpack4构建的一套基于微信公众号的前后端同构web应用 】

主要技术栈：koa2、vue2全家桶

打包构建工具：webpack4

第三方库：vant、axios、dubbo2.js、wechat-jssdk、dayjs等

***

# 开发

### 克隆项目

https://github.com/liuchanghao/vue-h5.git

### 进入项目目录

cd vue-h5

### 安装依赖
cnpm i

### 启动服务
npm run dev

http://127.0.0.1:8080/
